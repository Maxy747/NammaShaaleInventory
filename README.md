# 📦 NammaShaaleInventory  
### Smart School Inventory Management System  

NammaShaaleInventory is a full-stack inventory management system designed for schools to efficiently manage, monitor, and track inventory such as books, lab equipment, classroom assets, and administrative resources.  

Built to simplify school resource management with secure authentication, real-time stock updates, report generation, and admin controls.

---

## 🚀 Features  

### 👤 User Authentication  
- Secure Login & Registration  
- Role-based access (Admin / Staff / User)  
- Password encryption and validation  

### 📚 Inventory Management  
- Add, update, and delete inventory items  
- Categorize assets (Books, Lab Equipment, Furniture, Electronics, etc.)  
- Real-time stock quantity tracking  
- Item availability status  

### 📊 Reports & Analytics  
- Generate downloadable reports  
- Inventory usage summaries  
- Low stock alerts  
- Asset history tracking  

### 🛠️ Admin Dashboard  
- Manage users  
- View complete inventory database  
- Monitor stock movement  
- User activity overview  

### 📱 Additional Functionalities  
- QR Code integration for inventory identification  
- Responsive UI for desktop/mobile use  
- Search and filter system  
- Data storage and retrieval
